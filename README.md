# zambia-cholera-geospatial-modelling
This repository contains the data for replication of geospatial modeling work conducted on the Oct 2017-May 2018 cholera outbreak in Lusaka, Zambia to identify associated risk factors. 

It accompanies the paper "Cholera risk in Lusaka: A Geospatial Analysis to Inform Improved Water and Sanitation Provision" by Peter W. Gething, Sophie Ayling, Josses Mugabi, Odete Duarte Muximpua, George Joseph.

Corresponding author: Sophie Ayling sayling@worldbank.org
